## 💡 遊戲設計理念

1. **Refuse死記硬背**：以前自己學排版常常搞不懂主軸跟交差軸，所以特別把講義精華濃縮成 5 個「新手友善關卡」，幫大家快速補血。
2. **給你魚吃不如教你釣魚**：一開始直接給答案就不好玩了！遊戲內故意隱藏答案，而是提供「階梯式提示」，引導大家親自動手敲程式碼，印象才深刻。
3. **卡片動次動，即時呈現不盲猜**：代碼敲進去，右邊的畫布就會立刻跟著變動，還有自動檢查過關機制，破關畫面超療癒，絕對是新手的排版救星！

## 🏆 闖關任務與正確答案（攻略本）

本遊戲只使用核心的 Flexbox 屬性，以下是 5 個關卡的目標、提示與正確答案：

### 🎯 第一關：讓三張卡片橫排

* **任務描述**：預設的區塊元素是上下排列的。請使用屬性讓這三張卡片「橫向一排」吧！
* **正確答案**：

```css
flex-direction: row;

```

* **觀念解析**：flex-direction 用來控制主軸的方向，設定為 row 就會讓子元素從左到右橫向排列。

### 🎯 第二關：讓卡片水平置中

* **任務描述**：太棒了！接下來請讓這三張橫排的卡片，在畫布中「水平置中」對齊。
* **正確答案**：

```css
justify-content: center;

```

* **觀念解析**：justify-content 負責控制子元素在「主軸（目前是水平軸）」上的對齊方式，center 即為置中。

### 🎯 第三關：讓卡片垂直置中

* **任務描述**：現在卡片已經水平置中了，請再加把勁，讓卡片也「垂直置中」吧！
* **正確答案**：

```css
align-items: center;

```

* **觀念解析**：align-items 負責控制子元素在「交叉軸（目前是垂直軸）」上的對齊方式。當主軸是水平時，它就能做到垂直置中的效果。

### 🎯 第四關：讓主軸方向改成直向

* **任務描述**：轉個彎吧！請將主軸方向改成「直向排列」，讓卡片轉為垂直由上往下排。
* **正確答案**：

```css
flex-direction: column;

```

* **觀念解析**：將 flex-direction 改為 column 後，主軸會變成垂直方向，子元素就會像直欄一樣由上往下排列。

### 🎯 第五關：讓卡片太多時自動換行

* **任務描述**：大魔王來了！這裡有 5 張卡片，寬度已經超出容器了。請設定讓卡片在空間不足時「自動換行」。
* **正確答案**：

```css
flex-wrap: wrap;

```

* **觀念解析**：Flexbox 預設會強迫所有元素擠在同一行。設定 flex-wrap: wrap; 就能在空間不夠時，讓後續的卡片自動掉到下一行。

## 🛠️ 技術交底（專案架換）

本專案採用「單一檔案網頁（Single File Web Page）」結構開發，將 HTML、CSS 與 JavaScript 完整封裝在 index.html 之中。

* **HTML5**：建構遊戲的基本骨架（控制面板、畫布、卡片）。
* **CSS3**：負責遊戲整體的視覺美化，並動態接收使用者的輸入來改變樣式。
* **JavaScript (Vanilla JS)**：負責關卡切換邏輯、即時讀取 textarea 中的字串、寫入 DOM 元素樣式，並利用 window.getComputedStyle() 進行嚴格的過關條件驗證。
