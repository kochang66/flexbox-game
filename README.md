## 🏆 闖關任務與正確答案（攻略本）

本遊戲聚焦於最核心的 Flexbox 屬性，以下是 5 個關卡的任務目標、快速提示與正確答案：

### 🎯 第一關：讓倉鼠橫向排成一列

* **任務描述**：小倉鼠們預設是上下疊在一起的。請使用屬性讓這三隻倉鼠舒服地「橫向排成一列」吧！
* **快速提示**：設定主軸「橫向排列」→ `flex-direction: row;`
* **觀念解析**：`flex-direction` 用來控制 Flex 容器的主軸方向，設定為 `row` 就會讓子元素（小倉鼠）從左到右橫向排列。

### 🎯 第二關：讓倉鼠水平置中

* **任務描述**：太棒了！接下來請讓這三張橫排的倉鼠，在籠子裡「水平置中」對齊集合。
* **快速提示**：設定主軸「水平置中」→ `justify-content: center;`
* **觀念解析**：`justify-content` 負責控制子元素在「主軸（目前是水平軸）」上的對齊方式，`center` 即為置中。

### 🎯 第三關：讓倉鼠垂直置中

* **任務描述**：現在倉鼠已經水平置中了，請再加把勁，讓牠們在上下高度也「垂直置中」吧！
* **快速提示**：設定交錯軸「垂直置中」→ `align-items: center;`
* **觀念解析**：`align-items` 負責控制子元素在「交叉軸（目前是垂直軸）」上的對齊方式。當主軸是水平時，它就能做到垂直位置上的置中。

### 🎯 第四關：讓主軸方向改成直向排列

* **任務描述**：換個隊形！請將主軸方向改成「直向排列」，讓倉鼠們轉為垂直由上往下排成一縱隊。
* **快速提示**：設定主軸「直向排列」→ `flex-direction: column;`
* **觀念解析**：將 `flex-direction` 改為 `column` 後，主軸會切換為垂直方向，子元素就會像直欄一樣由上往下排列。

### 🎯 第五關：讓倉鼠太多時自動換行

* **任務描述**：大魔王來了！這次來了 5 隻大倉鼠，籠子塞不下啦。請設定屬性讓空間不足時牠們能「自動換行」到下一排。
* **快速提示**：設定「自動換行」→ `flex-wrap: wrap;`
* **觀念解析**：Flexbox 預設會強迫所有元素擠在同一行（不換行）。設定 `flex-wrap: wrap;` 就能在畫布空間不夠時，讓後續的倉鼠自動掉到下一行，避免擁擠。

## 🛠️ 技術交底與架構優化

本專案採用高度封裝的「單一檔案網頁（Single File Web Page）」結構開發，將 HTML5、CSS3 與 JavaScript (ES6+) 完美整合在 `index.html` 之中，便於部署與快速渲染。

* **HTML5**：建構遊戲的語意化骨架，並動態生成帶有 No. 角標的動態倉鼠 DOM 結構。
* **CSS3 視覺美化與特效**：
  * **黃色系色彩美學**：採用高質感「向日葵黃」到「葵瓜子金」的漸層文字特效（`linear-gradient` 搭配 `-webkit-background-clip`），大幅提升視覺完成度。
  * **動態滾動條（Scrollbar）生命週期管理**：開場選單與畢業畫面嚴格鎖定 `overflow: hidden;` 以維持視覺純淨度；遊戲關卡中動態釋放 `overflow-y: auto;`，確保不同裝置的使用者皆能流暢下滑操作。
* **JavaScript (Vanilla JS)**：
  * **即時編譯與驗證**：即時讀取 `textarea` 中的 CSS 字串並動態寫入樣式表，利用 `window.getComputedStyle()` 抓取瀏覽器底層解析後的實際樣式進行嚴格的條件判斷。
  * **DOM 動態重建技術**：在點擊「再玩一次」時，透過 JavaScript 徹底清除並動態重建初始化節點，強制瀏覽器 100% 重新載入無快取殘留的開場全螢幕常駐灑花動畫，解決傳統網頁動畫重複觸發失效的痛點。
